# Python Projects

A collection of Python scripts built for automation and practice. 

## Scripts

### organizer.py
Organizes files into folders based on file type.

### renamer.py
Bulk file renamer in a folder. || Change the "base_name" to your preference.

### photo_date_sorter.py
Bulk photo sorter. It sorts the photos in a library into two new directories ~ year/month/ 

### duplicate_finder.py
 It scans a folder (and its subfolders) and identifies duplicate files. Not by filename, but by actual content.
